# Movie Recommendation System

A movie recommendation web application built using Python and Flask.

## Features
- User login system
- Movie recommendation functionality
- Movie details page
- Responsive frontend using HTML, CSS, and JavaScript

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## Project Structure

```text
movie-recommender/
│
├── app.py
├── requirements.txt
├── templates/
├── static/
├── data/
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/KunwarShivangSingh/movie_recommendation_system.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```
